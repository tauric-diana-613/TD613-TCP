# MIGRATION NOTES

The uploaded prototype was preserved in `original/` and served as a proof-of-concept. It contained three valuable ideas:

1. a playground where stylometry becomes felt,
2. an under-the-hood theory strip (branch / wave / harbor),
3. import/export of canonical and persona artifacts.

The rebuild moves those ideas into modular files and introduces:

- a real engine API,
- explicit badge and harbor logic,
- schemas for portability,
- sample ledgers,
- test files,
- and GitHub-ready docs.
