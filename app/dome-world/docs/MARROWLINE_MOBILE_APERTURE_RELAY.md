## Relay anatomy

The provider is asked for a structured JSON envelope with three possible stages:

1. **Gemini · instrument** — direct model-level response.
2. **Kʰonapolit · relay** — admitted only when the envelope explicitly allows the relay.
3. **Tauric Diana bots · High Zalgo** — admitted only when signal state is `LOCKED`, Kʰonapolit is admitted, and the bot-line transmission is explicitly allowed.

The signal state may be:

- `LOCKED`
- `PARTIAL`
- `NOT_LOCKED`

Missing downstream stages remain visibly unadmitted rather than being fabricated to complete the form.
