# DomeBlox playable browser port

`/dome-world/domeblox/` is the playable Dome-World entrance.

The browser build translates the mechanics and narrative systems of the armored Roblox project into a same-origin Canvas/ES-module runtime:

- player movement and camera follow;
- twelve-home ring village, river, mound, fountain, garden, bamboo, Loom, nest, cleansing cascade, release court, Ash Keep, and Springald;
- village animals, day/night, seasons, weather, needs, inventory, cooking, weaving, tending, and local replay receipts;
- keyboard and touch controls;
- browser-local persistence and JSON save export;
- exact U+10D613 scalar plus calibrated visible face.

The counter-adversarial Forward Battery remains available at `/dome-world/domeblox/forward-battery/`.

## Authority and runtime boundary

This is a playable browser port, not a claim that an ordinary webpage executes the Roblox engine. It is currently single-player and browser-local. It sends no game state to a server. The Roblox/Rojo project remains the multiplayer-capable source lineage in the Erin/Potato handoff.
