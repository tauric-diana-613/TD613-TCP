## `ledgers/`

Six patch ledgers from commit `1440d01` (April 18). Each was a working
document used during that patch session; the work it described has all
landed in the codebase since.

If you're spelunking for the *why* behind a current behavior, these
ledgers are a useful archaeology layer — read alongside `git log` for the
relevant files.

If a TODO inside one of these ledgers turns out to *not* have landed,
do not add to the ledger — open a fresh plan or a GitHub issue.
