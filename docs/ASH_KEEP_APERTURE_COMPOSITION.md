## Purpose

Stretch 5 turns composition itself into a governed record.

The repository already contains Aperture v3.1 controlled sources, instrument ensembles, snapshot lattices, experiment runs, tomography receipts, Choir calibration bindings, and Hush intervention receipts. The renovation prevents those individually valid organs from being assembled by an unrecorded interface path.

```text
current Authority Context
→ controlled source
→ fixed instrument ensemble
→ snapshot lattice
→ experiment run
→ tomography receipt
→ receipt-bound Choir calibration
→ lifecycle-bound Hush intervention receipt
→ read-only presentation projection
```

Changing the order creates a held composition rather than a silent reinterpretation.
