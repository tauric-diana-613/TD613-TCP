## Purpose

Stretch 4 converts Hush from an adjacent derivative generator into a lifecycle-bound intervention instrument.

The packet asks:

> Can a current Rebuild-eligible case declare what must remain fixed, vary one or more named discourse dimensions, compare the returned candidates under matched Readers and controls, and preserve the result as an unkept candidate without silently acquiring release or transport authority?

## Constitutional binding

An intervention ensemble requires:

```text
current verified Authority Context
+ lifecycle permission HUSH_CANDIDATE
+ current Case Map digest
+ current Route Memory digest
+ current verified Rebuild receipt
+ source-text digest
+ fixed proposition obligations
+ fixed protected-literal policies
+ declared transformation dimensions
→ Hush Intervention Ensemble
```

Visibility never grants eligibility. `HUSH_CANDIDATE` remains gated at `REBUILD_ELIGIBLE`.

## Validation circuit

A receipt may become `INTERVENTION_ELIGIBLE` only when the packet carries:

1. an untampered ensemble;
2. the current matching Authority Context;
3. the current matching Rebuild receipt;
4. satisfied proposition obligations;
5. satisfied protected-literal policies;
6. a held source-drift posture;
7. repeated observed trials;
8. at least two declared Reader classes;
9. matched Reader sets and exact source/candidate digest parity;
10. a benign control;
11. a held-out trial;
12. transformation history;
13. resolved prompt-injection review;
14. provider-log parity whenever provider-draft mode was used.

The comparison remains componentwise. `universal_score = null`.
