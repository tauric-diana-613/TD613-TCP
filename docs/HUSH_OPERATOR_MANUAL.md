# TD613 Hush Operator Manual

## Purpose

Hush is a local authorship-pressure and custody review instrument. It helps an operator transform, review, and export text while preserving protected literals, event shape, claim limits, and safety warnings.

## What Hush Cannot Promise

Hush does not prove anonymity, untraceability, publication safety, legal sufficiency, platform outcome, or detector-proof authorship.

## Product Route

Use `/hush` as the primary product surface. The legacy `/adversarial-bench.html` route remains available for continuity.

## Operator Modes

- Preserve source register when speaker style matters.
- Stabilize source text when messy notes need clarity without whitening.
- Formalize only when explicitly requested.
- Target-register generation requires explicit target mode and review.

## Test Flight Protocol

Use synthetic or scrubbed samples first. Review hard blocks, warnings, target-register audit, release policy, export policy, and Recognition Field before carrying output.

## Stop Conditions

Stop when Hush reports literal loss, event-shape loss, certainty inflation, register erasure, target-register overreach, or export over-disclosure.
