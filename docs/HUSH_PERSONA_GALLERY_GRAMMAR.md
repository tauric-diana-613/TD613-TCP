# Hush Persona Gallery Grammar

## Card fields

Every persona card should show:

- persona name
- family
- story
- use when
- risk tell
- pressure warnings
- transform hints
- route class

## Route classes

- `field-persona-card`
- `lab-stress-card`
- `custody-lantern-card`
- `target-register-card`
- `forge-born-card`

## Rule

Persona theater must keep route warnings visible. The story invites play; the chips preserve review posture.
