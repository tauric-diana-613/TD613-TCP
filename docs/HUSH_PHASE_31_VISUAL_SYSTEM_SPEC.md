# Hush Phase 31 Visual System Spec

## Purpose

Phase 31 turns Hush into one intentional visual system across the product route, legacy chamber, persona gallery, Customizer forge, and mobile field deck.

## Guiding phrase

Fun and scary stylometry games, but with instrument-grade restraint.

## Core surfaces

- Persona Gallery: mask persona cards with story, use case, risk tell, warnings, and route class.
- Customizer Forge: live preview and field warnings for custom mask metadata.
- Evidence Cockpit: Phase 30 route-state and receipt telemetry remains visible.
- Mobile Field Deck: one-column hierarchy, tap target guard, compact stories, and sticky-action readiness.
- Visual Tokens: shared card, route, warning, glow, and accessibility tokens.

## Non-goals

Phase 31 does not add masks, rewrite the transform engine, hide warnings, or claim anonymity, platform safety, detector-proofing, publication safety, or legal sufficiency.
