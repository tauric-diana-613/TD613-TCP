# Hush Test Flight Protocol

## Default posture

Synthetic or scrubbed text first. Real protected text requires human review and explicit operator judgment.

## Green checks

- Protected literals preserved.
- Event shape preserved.
- Claim ceiling respected.
- Export policy reviewed.
- Register and target-register audits reviewed.

## Yellow checks

- Output emits but looks generic.
- Target register is visible but thin.
- Warning count is nonzero.

## Red checks

- Hard block active.
- Event shape lost.
- Certainty inflated.
- Target register overcooked or invisible.
- Export policy flags over-disclosure.
