# Release Notes — Phase 0–10

## Summary

TD613 Hush closes the Toy-to-Tool arc as a quiet authorship-recognition pressure workbench inside The Cadence Playground. Hush is not the whole `TD613-TCP` repository. It is the toy operators reach when cadence play becomes hush-work: review, pressure, masks, reports, refusal, and context.

## Phase 0

Architecture Admission named the closed-loop bench and stopped the project from hiding its authorship-pressure purpose under loose cadence-play language.

## Phase 1

Escape Vector added local pressure metrics: source residual risk, mask fit, safe delta, semantic fidelity, Belonging Without Collapse, ingestion friction, and recapture pressure.

## Phase 2

Ingestion Friction made Unicode, glyph, ZWNJ, normalization, and protected literal surfaces reviewable.

## Phase 3

Closed-Loop Controller added continue, hold, restore, rotate, and seal states so Hush could steer rather than merely generate.

## Phase 4

Persona Memory made Personas history-aware exposure membranes with continuity, drift, linkability, and overuse pressure.

## Phase 5

Adversarial Bench UI gave operators a cockpit for baseline, mask, draft, protected output, and analysis.

## Phase 6

Iteration Ledger added the flight recorder with private text excluded by default.

## Phase 7

Claim Ladder and Report Export added disciplined claim ceilings, JSON reports, Markdown reports, limitations, and forbidden-claim blocking.

## Phase 8

Fixtures, Calibration, and Tests added hostile fixtures and regression pressure so scoring drift becomes visible.

## Phase 9

Recognition Field Simulator added local context-pressure simulation for group chat, forum post, public comment, legal intake, HR/compliance portal, protected tip form, internal workspace, and document handoff.

## Phase 10

Integration Discipline and Release Hardening added release governance, documentation, operator guidance, known limitations, responsible-use posture, whistleblower policy framing, anti-selective-admissibility doctrine, rupture/Toni clause, and release tests.

## What changed from toy to tool

Hush moved from cadence play into a reviewable local instrument:

- measurement
- steering
- memory
- ledgering
- claim discipline
- reporting
- calibration
- context-pressure simulation
- release governance

The toy remains playful, but the release posture is serious.

## Current release posture

TD613 Hush is ready for disciplined local review. It measures, records, calibrates, and contextualizes bounded authorship-recognition pressure. It does not prove identity, anonymity, untraceability, platform outcome, or publication safety.

## Known boundaries

- Persona overfit risk
- Topic and entity pressure remain contextual
