# SAFETY MODEL

## Core principle

Recognition is not repair. Similarity is not authorship. Route pressure is a warning condition, not a verdict.

## Primary protections

- `mirror.off` as default anti-reflective safe passage when route pressure is high
- receipt generation before interpretive expansion
- provenance threshold checks before harbor reuse
- witness-load governor to prevent permanent externalization of archival burden

## Things TCP explicitly avoids

- automated authorship declarations
- crisis or clinical intervention claims
- auto-escalation into public exposure
- flattening uncanny residue into reassurance text
