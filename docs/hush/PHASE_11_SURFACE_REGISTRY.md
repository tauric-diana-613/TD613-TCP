# Phase 11 — Surface Registry

Status: implementation doctrine.

The Phase 11 surface registry names every drawer panel and assigns jurisdiction before the UI can render it.

## Authority classes

- `display-only`
- `gate-evaluated`
- `export-port`
- `private-local`
- `runtime-evidence`
- `boundary-review`

## Required surface fields

Each surface declares:

- surface id
- label
- source phase
- source modules
- authority class
- allowed actions
- forbidden actions
- raw exposure posture
- test coverage

## Registry principle

No anonymous surface. No orphan panel. No button without jurisdiction.

The registry is how Phase 11 preserves elegance without letting the drawer become decorative confusion.
