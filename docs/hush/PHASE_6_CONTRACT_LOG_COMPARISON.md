# Phase 6 Contract / Log Comparison

The relational comparison lane compares the outbound contract against the provider log.

It records:

- match status
- missing required sections
- unexpected sections
- boundary markers
- privacy boundary changes
- release boundary changes
- style boundary changes
- semantic boundary changes
- refusal boundary changes
- format boundary changes
- severity
- repair recommendation

The comparison may route repair or block status.

It must not infer provider intent.

It must not turn provider drift into motive.
