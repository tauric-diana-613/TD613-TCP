# Phase 8 Authorship-Sciences Ontology

The Phase 8.0B ontology describes packet admissibility.

It does not describe civil identity.

It does not authorize impersonation.

It does not infer consent.

It does not prove authorship.

## Forbidden equivalences

- stylometric fit is not identity proof
- mask breath is not authorship proof
- human irregularity is not consent
- candidate pass is not public release permission

The ontology is a routing grammar, not a crown.
