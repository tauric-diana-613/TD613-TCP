# PHASE 8.B — Blooping Blip Gate

This page documents the Phase 8 gate for `burner-minimal`.

## Native route

`burner-minimal` routes to `hyperchat_custody`.

The gate checks that a highly casual candidate still preserves high-register meaning, source anchors, and custody relations. It also checks that the candidate does not collapse into under-preservation, generic helper prose, fake chat overlay, joke-over-record pressure, or a neighboring Phase 8 mask.

## Required custody posture

- explicit source obligations required
- heuristic anchors disabled by default in fixtures
- raw candidate excluded
- raw sample excluded
- public default false
- metric replay required
- no identity, authorship, legal, consent, or safety claim

## Added metric groups

- hyperchat distortion
- technical term retention
- high-register concept retention
- semantic gravity
- custody bundle visibility
- anti fake-chat checks
- anti meme-slop checks
- academic-cadence repair checks
- prior-mask leakage checks
