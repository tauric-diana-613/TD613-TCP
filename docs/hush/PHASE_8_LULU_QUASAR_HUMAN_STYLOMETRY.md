# Phase 8.LQ — Lulu Quasar Gate

This page documents the Phase 8 gate for `quirky-orbit`.

## Native route

`quirky-orbit` routes to `blue_orange_relief`.

The gate models controlled comic relief with immediate custody return. The mask may use one bounded strange image, but the record must come back quickly and remain clearer than the joke.

## Required custody posture

- explicit source obligations required
- heuristic anchors disabled by default in fixtures
- raw candidate excluded
- raw sample excluded
- public default false
- one odd image in passing candidates
- repeated image risk blocked
- invented prop hoard blocked
- semantic drift blocked
- broken-English or alien-parody patterns blocked
- no identity, authorship, legal, consent, or safety claim

## Added metric groups

- blue-orange relief
- anchor return latency
- custody after quirk
- emotional gravity under comedy
- one-image governance
- anti-mascot checks
- anti-mush checks
- anti-parody checks
- source-idiolect displacement
- prior-mask leakage checks

## Non-claims

Lulu does not prove authorship, identity, truth, intent, legal protection, consent, safe release, non-attribution, or public readiness.

Lulu can relieve pressure, deform cadence, and preserve anchors through bizarre syntax under a metric envelope. Lulu cannot make the joke become evidence.
