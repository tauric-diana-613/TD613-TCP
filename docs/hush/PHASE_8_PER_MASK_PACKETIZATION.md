# PHASE 8 — Per-Mask Packetization Protocol

Phase 8 packetizes masks one at a time.

One mask.

One packet.

One hash surface.

One PR.

Phase 8.0 creates the shared bench that every mask must pass through before individual packet PRs begin.

## Phase 8.0 files

- `app/engine/hush-per-mask-packet.js`
- `app/engine/hush-mask-stylometric-calibration.js`
- `app/engine/hush-anti-slop-audit.js`

## Packet schema

```text
td613.hush.per-mask-packet/v1
```

## Non-goals

Phase 8.0 does not create a real per-mask packet file for any canonical mask.

Phase 8.0 does not batch-packetize the gallery.

Phase 8.0 does not promote extension masks.

Phase 8.0 does not make masks public-default.

Stylometry is the filter.

The packet is the kiln.

Imperfection is allowed only when source custody survives.

No mask becomes a person.
