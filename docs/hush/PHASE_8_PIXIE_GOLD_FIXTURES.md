# Phase 8 Pixie Gold Fixtures

Glitching Pixie is the first Phase 8.1 prototype candidate.

Before Pixie can receive an individual packet, Phase 8.0C provides a fixture bank:

- one known-good compressed custody fixture
- one polished fake-casual fixture
- one evidence-dropping fixture
- one over-glitch fixture
- one missing-candidate fixture

The fixture file is:

```txt
app/data/hush-phase8-fixtures/glitching-pixie-fixtures.js
```

The goal is not to make Pixie cute.

The goal is to prove that the gate can distinguish a passing compressed-custody candidate from generic polish, source damage, decorative glitch, and missing realization.
