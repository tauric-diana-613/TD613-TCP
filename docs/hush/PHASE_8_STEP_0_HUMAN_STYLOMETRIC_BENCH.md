# Phase 8.0 — Human Stylometric Calibration Bench

Step 0 prepares the bench before any single mask is sealed.

It adds:

- Phase 7 ready-mask queue intake
- per-mask packet builder
- replay validator
- stylometric calibration builder
- human imperfection ledger
- anti-slop audit
- Phase 9 handoff shell

No mask may become an individual packet until the bench exists.

## Calibration standard

The calibration surface records:

- cadence profile
- distribution profile
- prompt bench
- acceptance bounds
- source-custody requirements
- sample-reuse blockers
- limitations

It evaluates transformation behavior, not identity.

## Human imperfection ledger

The ledger allows bounded asymmetry and blocks fake mess.

Allowed irregularity must preserve evidence.

Forbidden noise includes factual damage, random typo seasoning, costume texture, sample wording reuse, and overfit catchphrase behavior.
