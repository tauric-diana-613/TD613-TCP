# Phase 9 Full Collision Matrix

Status: staged by local runner.

The full matrix expands dangerous-pair testing into a 13×13 cross-mask surface. Rows represent the source mask. Columns represent neighbor leakage pressure.

Each generated cell records packet class, packet id, mask A, mask B, collision risk, leakage direction, dominant failure mode, severity, centroid distance, repair recommendation, and source-obligation metrics.

The full matrix is generated by `scripts/run-hush-phase9-audit.mjs` and covered by `tests/hush-phase9-cross-mask-collision.test.mjs`.

Severity scale: 0 clean distinctness; 1 cosmetic overlap; 2 repair required; 3 hard blocker.

The matrix records local collision risk only. It does not grant public-release authority or governance authority.
