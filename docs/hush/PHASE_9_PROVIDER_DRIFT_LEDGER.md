# Phase 9 Provider Drift Ledger

Status: fixture-backed until live provider flights are captured.

Provider output is testimony. Provider output is not governance.

The ledger compares outbound contract expectations, stored inbound provider logs or fixtures, and final packet/export posture.

## Required fields

- provider model
- prompt version
- flight packet version
- outbound contract hash
- inbound log hash
- source hash
- reference hash
- mask id
- mask label
- candidate text hash
- preserved propositions
- dropped propositions
- new claims
- style operation
- style note
- risk flags

## Phase 9 fixture posture

The local suite includes one clean fixture and one drift fixture so contract comparison logic can run without quota-dependent provider calls.

Live provider drift remains deferred to deployed runtime flight evidence.
