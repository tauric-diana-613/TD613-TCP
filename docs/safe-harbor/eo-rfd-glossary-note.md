# EO-RFD Glossary Note

In TD613 Safe Harbor maintenance, UI, export, verification, and release-discipline contexts, `EO` is shorthand for the EO-RFD route conscience / context lane (interface-only).

It is not a governmental executive-order reference, an external office, or government authority.

Preferred labels:

- EO-RFD route context
- EO-RFD route-conscience hook
- EO-RFD observed packet route
- `td613:eo-route`

Short labels such as `EO hook` or `EO status` are acceptable only when the surrounding Safe Harbor context already makes EO-RFD clear.
