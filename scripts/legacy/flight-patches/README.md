# Legacy TD613 Flight patch scripts

These scripts are historical recovery references from the mobile Flight repair sequence.
They are intentionally quarantined here so active tests, builds, and GitHub Actions do
not rewrite `app/safe-harbor/td613-flight.html` as a side effect.

The committed Flight HTML is the live source of truth. If a legacy repair needs to be
reused, inspect the script manually and port the needed change into source deliberately.
